我的劳动仲裁申请书 | 大厂被裁维权2N第5篇

2026-06-02
其他

我，北京某大厂长视频平台P7软件工程师，入职刚满10年。3月底被通知裁员，我放弃了签《协商一致》赔N+1，选择让公司单方面解约，然后通过法律途径维权2N。

这篇分享我分享一下我的两份劳动仲裁申请书，希望里面的内容可以给大家提供新的思路。

为什么是两份申请书？

其中一份包含了非法解除劳动合同赔偿金（2N）、年终奖和部分加班费，由于公司的考勤信息不对员工开放，所以我根据手里的证据没办法计算10年中所有的加班费，所以第一份申请书只主张我有证据的这部分。

而另外一份主张的诉求是请求法院责公司令出示我过去的完整考勤记录，并按照完整考勤记录足额支付加班费，并支付股票损失。单独列在一个申请单里有几个原因：

  1. 请求出示考勤记录、请求支付股票损失不属于仲裁管辖范围，但由于劳动仲裁是劳动纠纷诉讼的前置程序，必须要申请劳动仲裁后拿到《不予受理通知书》才能去法院起诉。
  2. 仲裁不能有多个被申请人，我希望升级事态增加多个被申请人，这也对最终赔偿的支付提供保障。
  3. 因为公司一定会在仲裁败诉后通过上诉拖延支付时间，所以我计划提前通过该案件立案，等待后期并案。

注意事项

  1. 要注意区分“经济补偿”和“赔偿金”，如果你主张的是N这个在《劳动合同法》里叫“经济补偿”，而非法解除劳动合同主张的2N叫“赔偿金”。
  2. 加班费要拆分成“延时加班”、“休息日加班”和“法定假日加班”，延时加班的加班费为“加班时的月薪÷21.75÷8x延时加班小时数x1.5”，休息日加班的加班费为“加班时的月薪÷21.75÷8x延时加班小时数x2”，法定假日的加班费为“加班时的月薪÷21.75÷8x延时加班小时数x3”。
  3. 如果要去法院起诉，注意《劳动仲裁申请书》中的诉求要和《起诉状》中的诉求保持一致。

声明

本文均为本人真实经历，已提起仲裁和诉讼。需要特别说明的是：我不鼓励所有人都走单解流程，也不建议所有人都去劳动仲裁。每个人的实际情况不同，请根据自身情况慎重决策。如果你未来还希望在大厂工作，建议谨慎选择维权方式，因为仲裁和诉讼记录可能会影响你下一份工作的背调和入职。内容不构成法律建议，每个人情况不同，建议咨询专业律师。

如果你也在经历裁员纠纷，欢迎评论区交流。后续我会持续更新庭审过程、证据收集技巧、判决结果。

离开北京后，我们为什么选择武汉

2026-06-01
其他

在确定要离开北京之后，我们花了很多时间来讨论这个问题：如果要选一座国内城市落脚，它必须满足什么条件？

  1. 教育水平是第一位的。有了孩子之后，教育就是当下最重要的事情，虽然每个家长嘴上都说着快乐教育，但每当看到师资数据和升学率的时候，还是会毫不犹豫的用脚投票。
  2. 其次是医疗水平。人到中年，上有老下有小，我们现在是医院的常客。另一方面，也希望自己的晚年有保障。
  3. 生活品质和生活成本。北京在这方面是一个极端，极高的生活成本和完全与之不匹配的生活品质，所以我们希望我们的下一个目的地能达到两者的平衡。
  4. 出行便利度。对于一个不出去浪就浑身难受的人来说，这一点很重要——高铁、机场、国际航班一个都不能少。
  5. 最后是气候宜人。如果能选择，我希望生活在一个四季如春的地方，因为以前在东北，一年只能有半年的时间能出门，感觉生命都因此变短了。

备选城市有哪些，为什么没选？

大理

每个文艺青年心里都有一个大理。四季如春，苍山洱海，风光和气候是真的好。房价也比北京友好太多。

但我们还是把它排除了。医疗和教育是硬伤。孩子以后要考大学，不能全指望天赋。另外大理交通很不便利且成本高，出趟门基本都要从昆明转，长期来看不太现实。

对丁克或者退休养老的人来说，大理是天堂；但对我们这种带着孩子的家庭来说，可能不太适合。

深圳

深圳曾经是我们的首选。我们都是互联网行业，深圳和我们的职业更匹配，城市年轻、高效、有活力，我们甚至很多年前就布局了深圳户口。劝退我们的，是房价，深圳的房价全国第一。就算我们卖掉北京的房子去置换，也未必能换来同等的居住品质，而且还要背上更重的房贷。我们离开北京是为了活得轻松一点，不想换个地方继续还贷。

上海 / 广州

上海和广州我们也认真想过。这两个城市的教育资源和资料资源都是顶级的，生活水平丰俭由人，户口也相对好落实，但最关键的问题不是户口，而是我们不想再继续“辛苦工作”了。在北京十年，我们已经体验过一线城市的快节奏。到了上海广州，大概率还是会进入同样的轨道——高收入，高支出，高强度，少时间。我们想要的是另一种生活，不是换个城市的同一种活法。

杭州 / 青岛 / 成都

这三个城市，各有各的好。杭州有大厂，产业和我们匹配，房价比一线便宜但也不低。青岛有好的气候，靠海，适合生活。房价比一线城市便宜不少，生活成本友好得多。成都有缓慢的生活节奏和无数美食，是大家公认的“来了就不想走”的城市。
但我们始终缺乏一个行动的理由——没有亲人，没有明确的工作机会，只是为了“面朝大海春暖花开”难以让现在的我们为此付诸于行动。

为什么最终选了武汉？

说实话，武汉的硬伤很明显——气候。

武汉夏季可能要持续半年，温度常常达到四十多度，是中国著名的火炉。且因为武汉水多湿度大，会导致夏天湿热，冬天阴冷，而且武汉没有暖气，可以说这个纬度是国内冬天最冷的地方。从“气候宜人”这个标准来看，武汉非常的不合适。

但武汉有一个所有城市都没有的优势——亲人。

大芳是襄阳人，她的兄弟姐妹大多都在武汉，或者离武汉不远。对孩子来说，这意味着每年春节、暑假、国庆，不再是孤零零的一家三口，而是有一群亲戚可以走动；对大芳来说，这意味着她不再是远嫁到北方的异乡人，而是回到了离娘家最近的大城市。

除了气候，武汉在其他几个条件上都完全满足：

  • 教育资源全国一流。 武汉有80所高等院校，全国前三，7所211高校，包括武汉大学和华中科技大学。从幼儿园到大学，孩子基本可以不离开武汉。
  • 医疗资源也是全国顶尖。 同济、协和、中南医院、人民医院等多家三甲医院排名全国前列。全家老小的健康，在这里不用太担心。
  • 房价非常友好。 相比于五年前的高点武汉房价接近腰斩，卖掉北京的房子，在武汉可以买一套非常好的房子，剩下的钱也够过完下半辈子。
  • 生活成本可控。 武汉的物价水平比北京低一截，有了更多可支配的钱和时间，生活质量比在北京高了一大截。
  • 出行非常便利。 武汉是所谓的“九省通衢”，我惊奇的发现，从武汉出发坐高铁去”北上广深杭“都只要4个小时。

如果气候是唯一的代价，换来孩子有人陪、爱人有家人、老人有照应——这笔账，很值。

离开不是失败，而是一种选择。有人选风景，有人选机会，有人选亲人和烟火气。我们选择了后者。

离职前我收集了这6类证据 | 大厂被裁维权2N第4篇

2026-05-31
其他

我，北京某大厂长视频平台P7软件工程师，入职刚满10年。3月底被通知裁员，我放弃了签《协商一致》赔N+1，选择让公司单方面解约，然后通过法律途径维权2N。

这篇分享我在离职前偷偷收集的全部证据。如果你也打算维权，越早开始越好——因为权限随时会被收回。

注意事项

  1. 权限随时被收回：一旦被约谈，公司可能随时关闭你的代码、文档、聊天权限。所有证据尽量在第一次谈话前就备份好。
  2. 不要触发监控红线：公司会监控文件外发、U盘拷贝、网络上传。建议用个人设备（不要连公司WiFi）/相机拍摄拍照或录影，不要直接传文件，该设备要妥善保管不要清理原文件。
  3. 录音有规则：录音可以作为证据，但需要满足合法性要求。具体规则问律师或AI，这里不展开。
  4. 证据要保密：后期处理证据时注意脱敏，提交只给关键信息截屏的复印件，不要交出原文件。

我收集的6类证据

一、劳动关系证明（证明你在这家公司干过、干了多久）

  1. 劳动合同若干、转签协议2份→ 证明工龄连续计算
  2. 社保缴纳记录、纳税记录、工牌→ 辅助证明在职期间
  3. 聘用通知书、办理签证时候用到的收入证明→ 证明入职日期和薪资

二、违法裁员的铁证（证明公司单方面解约）

  1. 单方面解除劳动合同通知（最重要）
  2. 裁员约谈时的录音
  3. 岗位撤销通知邮件照片

三、薪资+股票+奖金证明（用于计算2N基数）

  1. 在职期间和离职后的银行代发流水
  2. 在职期间和离职后的工资明细单
  3. 系统中过往合同工资的照片
  4. 股权系统中未归属股权的截图

四、加班证据（分两类：客观记录+强制指令）

客观加班记录：

  1. 十年内所有的代码提交记录
  2. 考勤系统的照片（部分）
  3. 加班打车记录的截图
  4. 故障复盘邮件照片（里面包含加班处理故障的时间线）

强制加班指令（更关键）：

  1. 主管转发的996相关邮件照片
  2. 996动员会的会邀邮件照片
  3. 聊天软件中主管/上级主管关于项目进度、周末加班的讨论照片

加班证据特别提醒：

代码提交记录、打车截图只能证明你“在那个时间工作过”，不能直接证明是“公司安排加班”。最有力的证据是：领导在工作群里要求“大家周末赶一下进度”、邮件通知“本周六正常上班”、或者你主动请示加班后领导同意的聊天记录。

五、工作内容和绩效证明

  1. 系统中团队和主管的变更记录的视频
  2. 系统中历年绩效和主管评价的视频（反驳公司说你“不胜任”）

这些证据支撑了我的哪些诉求

  1. 违法解除劳动合同赔偿金2N
  2. 一部分加班费
  3. 最近一年的年终奖
  4. 公司违约导致的股票损失

我没来得及收集的

  1. 更全面的工作内容记录
  2. 过往股票授予协议书（在股权系统里，权限被收后无法查看）
  3. 一些口头传达的强制加班证据（只有自己笔记，缺少录音）
  4. 十年间完整的考勤记录（已申请法院责令公司提供）

提醒大家：如果可能，尽早把这些也存下来。我会继续通过法律途径申请公司提交。

通用取证建议

不同工种、薪资结构不同，我的清单不适用于所有人，请根据自己情况调整。

重要技巧：去“中国裁判文书网”搜你公司或同行业的判决书，里面会详细列出法院认可的证据类型和判决理由——这是免费的法律老师。

声明

本文均为本人真实经历，已提起仲裁和诉讼。需要特别说明的是：我不鼓励所有人都走单解流程，也不建议所有人都去劳动仲裁。每个人的实际情况不同，请根据自身情况慎重决策。如果你未来还希望在大厂工作，建议谨慎选择维权方式，因为仲裁和诉讼记录可能会影响你下一份工作的背调和入职。内容不构成法律建议，每个人情况不同，建议咨询专业律师。

如果你也在经历裁员纠纷，欢迎评论区交流。后续我会持续更新庭审过程、证据收集技巧、判决结果。

我被违法辞退的全过程 | 大厂被裁维权2N第3篇

2026-05-30
其他

我，北京某大厂长视频平台P7软件工程师，入职刚满10年。3月底绩效面谈时被主管通知裁员。我放弃了签《协商一致》拿N+1，选择让公司发起单方面解约，然后走法律途径维权2N。

下面是我从被通知裁员到拿到《单方面解除劳动合同通知》的完整时间线，分享出来供大家参考。

先划重点：三个权限回收时间点

整个过程有三个关键时间点公司会回收系统权限，务必在此之前收集好证据：

  • 第一次回收（约谈当天）：工作文档、代码平台权限被收回
  • 第二次回收（单解流程确认后）：聊天软件、所有系统权限被收回
  • 第三次回收（LastDay次日凌晨）：办公区权限被收回、聊天软件自动退群

这里要特别说明，每家公司回收权限的流程不同，有些厂会在和你谈话的同时收回所有权限，本文提到的只是我前司在这次裁员中使用的流程，不保证一直有效。

注意事项：

  • 和主管、HR正式谈话时，不要说对自己不利的话，因为他们全程录音。
  • 做好自己录音的准备。录音作为庭审证据有特定规则，可以问AI或律师。
  • 录音设备放兜里别乱动，否则听不清（别问我是怎么知道的）。
  • 很多企业会监控文件外发、网络访问，取证要有技巧，千万别触发政策红线。

详细时间线

3月25日

10:00 团队主管绩效面谈，被告知裁员决定。

10:35 HR约谈，告知裁员政策：

  • 签《协商一致解除合同协议书》→ 赔 N+1
  • 超过4月9日不签 → 公司发起单方面解约 → 只赔 N

HR明确表示：没有任何谈判空间。

12:26 收到《岗位撤销通知》邮件

大致内容：因公司业务调整，自3月25日撤销您所在岗位。附三个转岗链接，如4月9日前未成功转岗，公司将与您协商解除劳动合同。同时通知：自3月26日起对您无工作安排，开始休假。公司会暂对您的工作账号进行调整，以确保数据安全。

18:00 第一次回收权限（工作文档、代码平台无法访问）

注：这个操作是HR手动发起的，随时可能发生，收集证据要趁早。 此时内网论坛、HR系统、邮件、聊天软件等仍可正常访问。

4月1日

今年归属的股票（RSU）已锁定。

4约6日

股票到账。

4月8日

HR二次约谈，我明确表示拒绝签署《协商一致解除合同协议书》。

4月9日

11:58 第二次回收权限（聊天软件 + 所有系统权限）

HR在聊天软件上发来消息：如3月25日沟通，因业务调整您的岗位已被撤销。应数据安全考虑，安全部门即将回收您的工作权限。后面附HR联系方式。 点开消息后几秒钟，所有系统和软件全部自动登出，仅保留办公区出入权限。

17:02 公司发出《单方面解除合同通知》

由公司HRSSC团队通过EMS寄到《劳动合同》中的通讯地址。

4月10日

18:35 签收《单方面解除合同通知》

4月11日

办理离职手续（此时还没有LastDay，流程仍在进行）：

  • 归还办公电脑、显示器
  • 赔偿丢失的测试手机
  • 还回接线板等行政借用品
  • 在安全同学协助下，将电脑里的个人照片导出到U盘
  • 从行政要了纸箱打包个人物品
  • 将用办公经费购买的无归属电子设备整理好放到工位边柜子里，拍照留存，避免后续被说贪公司便宜

4月15日（LAST DAY）

9:16 个人邮箱收到电子版离职证明（邮箱是《劳动合同》中留的那个）。

4月16日

00:00 自动退群，办公区门禁权限失效（第三次权限回收）。

4月30日

赔偿金 N、4月工资、股票分红到账。

收到半个月工资，因休完了今年的陪伴假，被扣了7000元。 另外发现停车费没有退还，后面会联系行政部门退还。

5月6日

过完五一小长假，拿着上个月准备好的材料，打扮的漂漂亮亮，开开心心的去仲裁委。

声明

来北京奋斗12年，我们决定离开了

2026-05-29
其他

一年前，我和大芳认认真真定下了一个deadline和一个详细的计划。

那个时候，我们不知道未来要去哪座城市，也没想好以后做什么。

但有一件事无比确定：

我们不想再这样活下去了，我们要离开北京。

1. 严重的内卷，和怎么都够不着的幸福感

不只是互联网行业，北京的各行各业，都缺少对个人生活最基本的尊重。企业只希望你无休止地工作，打工人被裹挟着只能不停地赚钱，不敢停，也不能停。

在这种氛围下，加班成了默认配置。我身边的大多数人，都很难尽到做父母的责任——要么一个人常年在北京工作，家人远在老家；要么夫妻俩都晚上八九点才到家。至少有一人处于这种状态，有时候两人都是，孩子基本上都是老人在带。

大芳负责公司的海外业务，情况更严重。她经常出差，甚至哺乳期也被安排出国。平时在家，也是不分白天黑夜地开会。孩子经常连续好几天见不到妈妈。

在这种畸形的环境里，人会格外缺乏安全感。多数人不敢停下脚步，因为只要一停下来，就会被贴上“不够努力”的标签，逐步从核心岗位上被淘汰。更残酷的是，即使你不停下来，到了一定年龄，或者遇上公司的一阵风吹草动，一样随时可能被淘汰出局。

程序员这个岗位，更是如此——不够听话，被裁；不够努力，被裁；年龄大了，被裁；创新项目失败了，被裁；年轻人更便宜了，被裁；AI提升了效率，还是被裁。

对于多数人来说，有能力+够努力，也打破不了这个魔咒，而且还要付出惨痛的代价。

2. 不公平的资源分配

北京还有一个让人特别难受的地方：很多基本的生活资源，都不能被公平分配。

先说户口。你要么名校毕业、一入职就拿下“投名状”，要么积分落户、被绑定在这里工作几十年。没有户口，你为这座城市付出再多，也没办法公平地享受教育资源。最离谱的是，如果你在北京工作，房子买在郊区，积分反而有额外加分——我实在想不出这背后的合理逻辑。

再说车牌。一张新能源车牌，排队十年是常态。我有很多同事油牌抽了十几年还没中。放在全世界任何一个地方，这都难以想象。而且，获得的排队和抽签的资格，门槛本身就很高。

3. 离谱的房价

前几年北京房价高得离谱，最近虽然降了不少，但对很多年轻人来说，依然高不可攀。

高房价不仅影响你能不能在这里舒适地生活，也影响那些小创业者能不能活下来。就拿餐饮来说，我观察到北京小餐饮店的生存时间短到离谱。有些地方，疫情之后每年要换掉至少两波门店。能活下来的，百里挑一。

还在努力挣扎的，就只剩下疲惫的打工人。

4. 恶劣的生活品质

内卷、资源不公、房价高企——三者叠加，很难有什么舒适的生活品质。

你要么根本没有时间生活，要么因为高房价住到一个特别远的地方，每天通勤耗掉几个小时。又或者，为了让孩子上一个好一点的学校，挤进所谓的“学区老破小”里。绝大多数人，都被困在这几种困境之中，动弹不得。

我们将要去什么样的地方？

离开不是为了逃避，而是为了去一个更对的地方。

我们的标准很简单：基础资源要有保障，孩子上学不用愁；孩子可以幸福快乐地成长，放学后能见到父母，周末有很多事情可以做；生活成本也要可控，不用把一辈子压在一套老破小上。

离开不是失败，而是一种选择。

有人选择留下继续拼，有人选择离开换一种活法。但人生的长度是有限的，我们选择增加人生的“厚度”。没有对错，只有适不适合。
