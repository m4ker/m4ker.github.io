北京卖房亏了230万，全程复盘

2026-06-09
其他

2025年11月，在决定离开北京半年之后，我们开始计划出售我们的房子。

房子是2020年10月买的。当时北京楼市还处于高点，我们买入的价格是755万，后期改造陆陆续续花了超过30万，总成本约785万。户型是6-7层的复式，面积偏大，有两个露台，在周边非常稀缺，但致命的缺陷是没有电梯。

2025年4月，同户型有一套成交价645万。但那时我还没开始卖房，也没有密切关注市场——实际上，就在我挂牌前的那个月，北京楼市经历了一次大跌，跌幅差不多20%。这个信息，是我后来才补上的，也给后面整个过程埋下了隐患。

TIP1：在房价下行的情况下，如果已经定下了卖房计划，越早卖亏越少。我这个房子，如果决定离开的时候就卖，能少亏100万。

一、最初的卖房策略

我明白市场有涨有跌，所以并不纠结于“房价下行期卖房合不合适”的问题。我制定的策略是：以大概每月一次的频率降价，降幅大于市场价下跌的幅度，直到把房子卖出去为止。拉长销售周期的目的是避免“曝光不足”导致成交价过低，风险在于销售周期拉长后，可能要继续承受房价继续下跌的损失。
我和之前认识的中介简单沟通后，分别在L、W、M上架了房源。挂牌价在中介的建议下暂定680万——虽然2025年4月同户型成交645万，但我的房子因为户型稀缺、装修投入大，我当时觉得680万是合理的。

二、漫长的降价过程

起初的一个月，来看房的人寥寥无几。最让我烦恼的是，每周L的商圈经理都要给我打一个电话，想办法让我大幅降价——他想让我降到550万左右。由于我已经定好了自己的降价节奏，所以一直没有同意。

TIP2：注意甄别中介的“骚扰电话”和“逼单策略”。有的经纪人为了压价，说话很难听。有些话确实反映了市场现实，但有些纯粹是为了打击你的心理防线。有道理的就听听，没道理的就别理，或者直接怼回去。经纪人是可以换的，别惯着他，别让他们影响到自己还有家人的情绪。

这个过程中，也一直出现一些买家抱着捡漏的心态来出价，大多数在500-550万之间。因为这种情况太多了，我和各家中介定下了一个规则：预算低于600万的买家暂时不见。那时候，我们的心理预期价是600万。

第一次降价（2026年1月2日）

在2个月的观察之后，我们发现真实的市场情况和我们的心理预期差距较大，所以大幅降低了心理预期，我们将挂牌价下调到660万，我们的心理价位也调整到了580万。

TIP3：要尽快摸清真实成交区间：不要因为不敢面对现实而拖延。拖得越久，可能亏得越多。

第二次降价（2026年3月20日）

为了不错过所谓的”小阳春“，我们决定把挂牌价下调到618万，心理价位仍然是580万。降价立刻见效，当天下午就来了三伙人来看房，之后的一周内中介陆陆续续带来了很多客户，其中也包括我们的“一号买家”。

三、一号买家的拉锯战

2026年4月8日

我们第一次和意向买家去M店里面谈，也第一次认清了买家的出价逻辑——对于这种流动性比较差的房产，买家多倾向于出一个比市场预估价格低很多的价格，一号买家的出价是从500万开始的，经过几次沟通只加到了不到510万。

2026年4月9日

一早就收到了M经纪人的长篇微信消息，经过了几轮沟通，经纪人表示一号买家愿意出价到520万。

2026年4月10日（第三次降价）

我们将挂牌价下调到598万，此时我们的心理价位还是580万。这次调价其实是对一号买家出价的回应，我们要加快进程。

2026年4月13日

晚上，一号买家带着孩子和老公登门拜访，在经过了将近一个小时的沟通后，把出价调整到540万，起初我坚持按照我的节奏继续降价，但在和大芳商量之后我们决定给出一定的诚意，将报价调整到560万，但并未收到对方答复。

四、终极行动

2026年4月17日（第四次降价）

我们联系了L做自媒体的销冠来家里拍视频，并进行了第四次降价（也是最后一次）：挂牌价下调至568万。目的很明确：快速找到愿意基于540万往上出价的买家，尽快成交；如果找不到，就以540万这个价成交。

TIP4：如果你的终极目的是尽快卖掉，就要接受可能亏更多。

2026年4月18日

上午L的销冠来拍视频，他做的视频标题是“一周必卖”。他非常准确地判断了市场：这一波推广和降价，是为了让之前那些关注了很久但没出价的潜在买家尽快行动。从这一天开始，看房的人络绎不绝，每天至少三五波，价格真的降到了大家的心理预期。但即使我们已经明确告知有540万买家存在，仍然有一些买家出价500-520万等着捡漏。

TIP5：不要等着中介帮你做推广，主动要求中介帮你联系，也要主动去自媒体平台上找流量，主动联系一些粉丝多的房产自媒体来拍视频。
TIP6：尽早找到靠谱的销冠级中介：通过沟通快速了解他的水平和销售策略，听取靠谱的建议可以少走很多弯路。

五、二号和三号买家

2026年4月20日

L带来了二号买家。上午约我们晚上去店里谈。中午又出现了三号买家——一位住在同小区的先生。因为我们家的户型在小区里非常稀缺，站在楼下就能定位到。他通过物业要到了我的联系方式，直接上门沟通。他一见面就和我说：“我关注你家房子很久了，之前价格有点高所以一直不好意思来砍价，昨天晚上看到你降价了，我差点当时就过来找你。”我和他说明当前情况后，他爽快出价550万，并说明了不通过中介交易的方案。为了增加自己的可信度，他给我看了他手机里不计其数的绿植照片，还给我看了他各个银行的余额。他说他想给他的绿植找个好地方。

如果不通过中介，双方各省五万五中介费，而且我了解到如果要拿定金还房贷，中介还要额外收费。所以三号买家的550万（无中介）相当于中介带来买家出价556万。如果要和二号买家继续谈，二号买家的出价必须超过556万。我们把这个情况和L的经纪人、二号买家说明后，把底线定在了560万。但二号买家坚持只出550万，双方没见面就僵持了两小时，最后二号买家放弃。

TIP7：真正进到决赛圈的买家，都是真心喜欢这种户型、能接受没电梯的缺点、常年在望京工作或生活、对周边房价很了解的人。他们愿意为稀缺属性和缺点买单。

2026年4月23日

我从武汉验收装修回来，第一时间去了L门店，约二号买家第二次谈判。我当面把所有情况和定价逻辑都解释了。但她和老公沟通后仍然坚持550万。谈判持续了约一个小时，我们主动退出。

谈判过程中，二号买家和我们分享了她听了经纪人的建议快速降价卖房的经历，还说由于感觉自己通过经纪人卖房亏了钱，希望再通过经纪人买房捡个漏。

TIP8：中介的降价建议要谨慎对待：专业中介对成交价区间的评估是准确的，但他们倾向于让你降更多，以便低成本快速成交。他们会以各种理由要你降价，甚至拿推广资源来施压。降价幅度和心理价位的调整，一定要自己算清楚。记住，中介的目标是快速成交，不是帮你卖高价。

谈判结束后我马上联系了三号买家，说明了二号买家的情况，和他约在第二天沟通交易细节。

2026年4月24号

我花了半天时间和三号买家沟通交易细节，签订购房合同，收到定金30万元。在沟通期间，L的经纪人还试图加价截胡——他们认为550万是我们最终的心理价位，甚至有人在楼下蹲守三号买家。

2026年5月21号

在收到银行提前还款审批通过的短信后，我和三号买家一同去了代办公司进行网签，并收到第二部分房款70万元用于还贷。

第一次和三号买家见面的时候，原定家具家电是不留下的，只留定制家具和空调，拟合同时我再一次和他确认家具的细节，他说如果能留下他就直接搬进来。我考虑了片刻，家里的每一件家具都精心挑选和搭配的，有些甚至替换过多次，如果我都变卖掉应该值个大几万块钱，但留在这才更有价值。

所以，我决定把家具都留下，也包括几盆养了五年的绿植，和鱼缸里养了两年的鱼。

我只是希望，让这里一切的美好持续下去。

它们只是换了新主人而已。

我的劳动仲裁申请书 | 大厂被裁维权2N第5篇

2026-06-02
其他

我，北京某大厂长视频平台P7软件工程师，入职刚满10年。3月底被通知裁员，我放弃了签《协商一致》赔N+1，选择让公司单方面解约，然后通过法律途径维权2N。

这篇分享我分享一下我的两份劳动仲裁申请书，希望里面的内容可以给大家提供新的思路。

为什么是两份申请书？

其中一份包含了非法解除劳动合同赔偿金（2N）、年终奖和部分加班费，由于公司的考勤信息不对员工开放，所以我根据手里的证据没办法计算10年中所有的加班费，所以第一份申请书只主张我有证据的这部分。

而另外一份主张的诉求是请求法院责公司令出示我过去的完整考勤记录，并按照完整考勤记录足额支付加班费，并支付股票损失。单独列在一个申请单里有几个原因：

  1. 请求出示考勤记录、请求支付股票损失不属于仲裁管辖范围，但由于劳动仲裁是劳动纠纷诉讼的前置程序，必须要申请劳动仲裁后拿到《不予受理通知书》才能去法院起诉。
  2. 仲裁不能有多个被申请人，我希望升级事态增加多个被申请人，这也对最终赔偿的支付提供保障。
  3. 因为公司一定会在仲裁败诉后通过上诉拖延支付时间，所以我计划提前通过该案件立案，等待后期并案。

注意事项

  1. 要注意区分“经济补偿”和“赔偿金”，如果你主张的是N这个在《劳动合同法》里叫“经济补偿”，而非法解除劳动合同主张的2N叫“赔偿金”。
  2. 加班费要拆分成“延时加班”、“休息日加班”和“法定假日加班”，延时加班的加班费为“加班时的月薪÷21.75÷8x延时加班小时数x1.5”，休息日加班的加班费为“加班时的月薪÷21.75÷8x延时加班小时数x2”，法定假日的加班费为“加班时的月薪÷21.75÷8x延时加班小时数x3”。
  3. 如果要去法院起诉，注意《劳动仲裁申请书》中的诉求要和《起诉状》中的诉求保持一致。

声明

本文均为本人真实经历，已提起仲裁和诉讼。需要特别说明的是：我不鼓励所有人都走单解流程，也不建议所有人都去劳动仲裁。每个人的实际情况不同，请根据自身情况慎重决策。如果你未来还希望在大厂工作，建议谨慎选择维权方式，因为仲裁和诉讼记录可能会影响你下一份工作的背调和入职。内容不构成法律建议，每个人情况不同，建议咨询专业律师。

如果你也在经历裁员纠纷，欢迎评论区交流。后续我会持续更新庭审过程、证据收集技巧、判决结果。

离开北京后，我们为什么选择武汉

2026-06-01
其他

在确定要离开北京之后，我们花了很多时间来讨论这个问题：如果要选一座国内城市落脚，它必须满足什么条件？

  1. 教育水平是第一位的。有了孩子之后，教育就是当下最重要的事情，虽然每个家长嘴上都说着快乐教育，但每当看到师资数据和升学率的时候，还是会毫不犹豫的用脚投票。
  2. 其次是医疗水平。人到中年，上有老下有小，我们现在是医院的常客。另一方面，也希望自己的晚年有保障。
  3. 生活品质和生活成本。北京在这方面是一个极端，极高的生活成本和完全与之不匹配的生活品质，所以我们希望我们的下一个目的地能达到两者的平衡。
  4. 出行便利度。对于一个不出去浪就浑身难受的人来说，这一点很重要——高铁、机场、国际航班一个都不能少。
  5. 最后是气候宜人。如果能选择，我希望生活在一个四季如春的地方，因为以前在东北，一年只能有半年的时间能出门，感觉生命都因此变短了。

备选城市有哪些，为什么没选？

大理

每个文艺青年心里都有一个大理。四季如春，苍山洱海，风光和气候是真的好。房价也比北京友好太多。

但我们还是把它排除了。医疗和教育是硬伤。孩子以后要考大学，不能全指望天赋。另外大理交通很不便利且成本高，出趟门基本都要从昆明转，长期来看不太现实。

对丁克或者退休养老的人来说，大理是天堂；但对我们这种带着孩子的家庭来说，可能不太适合。

深圳

深圳曾经是我们的首选。我们都是互联网行业，深圳和我们的职业更匹配，城市年轻、高效、有活力，我们甚至很多年前就布局了深圳户口。劝退我们的，是房价，深圳的房价全国第一。就算我们卖掉北京的房子去置换，也未必能换来同等的居住品质，而且还要背上更重的房贷。我们离开北京是为了活得轻松一点，不想换个地方继续还贷。

上海 / 广州

上海和广州我们也认真想过。这两个城市的教育资源和资料资源都是顶级的，生活水平丰俭由人，户口也相对好落实，但最关键的问题不是户口，而是我们不想再继续“辛苦工作”了。在北京十年，我们已经体验过一线城市的快节奏。到了上海广州，大概率还是会进入同样的轨道——高收入，高支出，高强度，少时间。我们想要的是另一种生活，不是换个城市的同一种活法。

杭州 / 青岛 / 成都

这三个城市，各有各的好。杭州有大厂，产业和我们匹配，房价比一线便宜但也不低。青岛有好的气候，靠海，适合生活。房价比一线城市便宜不少，生活成本友好得多。成都有缓慢的生活节奏和无数美食，是大家公认的“来了就不想走”的城市。
但我们始终缺乏一个行动的理由——没有亲人，没有明确的工作机会，只是为了“面朝大海春暖花开”难以让现在的我们为此付诸于行动。

为什么最终选了武汉？

说实话，武汉的硬伤很明显——气候。

武汉夏季可能要持续半年，温度常常达到四十多度，是中国著名的火炉。且因为武汉水多湿度大，会导致夏天湿热，冬天阴冷，而且武汉没有暖气，可以说这个纬度是国内冬天最冷的地方。从“气候宜人”这个标准来看，武汉非常的不合适。

但武汉有一个所有城市都没有的优势——亲人。

大芳是襄阳人，她的兄弟姐妹大多都在武汉，或者离武汉不远。对孩子来说，这意味着每年春节、暑假、国庆，不再是孤零零的一家三口，而是有一群亲戚可以走动；对大芳来说，这意味着她不再是远嫁到北方的异乡人，而是回到了离娘家最近的大城市。

除了气候，武汉在其他几个条件上都完全满足：

  • 教育资源全国一流。 武汉有80所高等院校，全国前三，7所211高校，包括武汉大学和华中科技大学。从幼儿园到大学，孩子基本可以不离开武汉。
  • 医疗资源也是全国顶尖。 同济、协和、中南医院、人民医院等多家三甲医院排名全国前列。全家老小的健康，在这里不用太担心。
  • 房价非常友好。 相比于五年前的高点武汉房价接近腰斩，卖掉北京的房子，在武汉可以买一套非常好的房子，剩下的钱也够过完下半辈子。
  • 生活成本可控。 武汉的物价水平比北京低一截，有了更多可支配的钱和时间，生活质量比在北京高了一大截。
  • 出行非常便利。 武汉是所谓的“九省通衢”，我惊奇的发现，从武汉出发坐高铁去”北上广深杭“都只要4个小时。

如果气候是唯一的代价，换来孩子有人陪、爱人有家人、老人有照应——这笔账，很值。

离开不是失败，而是一种选择。有人选风景，有人选机会，有人选亲人和烟火气。我们选择了后者。
