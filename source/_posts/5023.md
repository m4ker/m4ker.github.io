> 我，北京某大厂长视频平台P7软件工程师，入职刚满10年。3月底被通知裁员，我放弃了签《协商一致》赔N+1，选择让公司单方面解约，然后通过法律途径维权2N。

这篇分享我在离职前偷偷收集的全部证据。如果你也打算维权，越早开始越好——因为权限随时会被收回。

### 注意事项

# 权限随时被收回：一旦被约谈，公司可能随时关闭你的代码、文档、聊天权限。所有证据尽量在第一次谈话前就备份好。
# 不要触发监控红线：公司会监控文件外发、U盘拷贝、网络上传。建议用个人设备（不要连公司WiFi）/相机拍摄拍照或录影，不要直接传文件，该设备要妥善保管不要清理原文件。
# 录音有规则：录音可以作为证据，但需要满足合法性要求。具体规则问律师或AI，这里不展开。
# 证据要保密：后期处理证据时注意脱敏，提交只给关键信息截屏的复印件，不要交出原文件。

### 我收集的6类证据

#### 一、劳动关系证明（证明你在这家公司干过、干了多久）

# 劳动合同若干、转签协议2份→ 证明工龄连续计算
# 社保缴纳记录、纳税记录、工牌→ 辅助证明在职期间
# 聘用通知书、办理签证时候用到的收入证明→ 证明入职日期和薪资

#### 二、违法裁员的铁证（证明公司单方面解约）

# 单方面解除劳动合同通知（最重要）
# 裁员约谈时的录音
# 岗位撤销通知邮件照片

#### 三、薪资+股票+奖金证明（用于计算2N基数）

# 在职期间和离职后的银行代发流水
# 在职期间和离职后的工资明细单
# 系统中过往合同工资的照片
# 股权系统中未归属股权的截图

#### 四、加班证据（分两类：客观记录+强制指令）

客观加班记录：

# 十年内所有的代码提交记录
# 考勤系统的照片（部分）
# 加班打车记录的截图
# 故障复盘邮件照片（里面包含加班处理故障的时间线）

强制加班指令（更关键）：

# 主管转发的996相关邮件照片
# 996动员会的会邀邮件照片
# 聊天软件中主管/上级主管关于项目进度、周末加班的讨论照片

加班证据特别提醒：

代码提交记录、打车截图只能证明你“在那个时间工作过”，不能直接证明是“公司安排加班”。最有力的证据是：领导在工作群里要求“大家周末赶一下进度”、邮件通知“本周六正常上班”、或者你主动请示加班后领导同意的聊天记录。

#### 五、工作内容和绩效证明

# 系统中团队和主管的变更记录的视频
# 系统中历年绩效和主管评价的视频（反驳公司说你“不胜任”）

### 这些证据支撑了我的哪些诉求

# 违法解除劳动合同赔偿金2N
# 一部分加班费
# 最近一年的年终奖
# 公司违约导致的股票损失

### 我没来得及收集的

# 更全面的工作内容记录
# 过往股票授予协议书（在股权系统里，权限被收后无法查看）
# 一些口头传达的强制加班证据（只有自己笔记，缺少录音）
# 十年间完整的考勤记录（已申请法院责令公司提供）

提醒大家：如果可能，尽早把这些也存下来。我会继续通过法律途径申请公司提交。

### 通用取证建议

不同工种、薪资结构不同，我的清单不适用于所有人，请根据自己情况调整。

重要技巧：去“中国裁判文书网”搜你公司或同行业的判决书，里面会详细列出法院认可的证据类型和判决理由——这是免费的法律老师。

### 声明

本文均为本人真实经历，已提起仲裁和诉讼。需要特别说明的是：我不鼓励所有人都走单解流程，也不建议所有人都去劳动仲裁。每个人的实际情况不同，请根据自身情况慎重决策。如果你未来还希望在大厂工作，建议谨慎选择维权方式，因为仲裁和诉讼记录可能会影响你下一份工作的背调和入职。内容不构成法律建议，每个人情况不同，建议咨询专业律师。
